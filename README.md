# lone-approval-analysis

Utilised Python libraries (NumPy, Pandas, Matplotlib, Seaborn, Plotly, Scikit-learn) to analyse bank loan data, exploring relationships between applicant information, loan details, and approval decisions.
Implemented algorithms to uncover quantitative relationships, providing insights to guide credit decisions and improve access to credit.
